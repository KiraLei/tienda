# KIRA STORE

_Tienda virtual creada para ventas de productos ecologicos_
## Comenzando 🚀
--------------------------------------------------------------------------------------------------------------
_Para poder tener una copia del proyecto solo necesitaras hacer los siguientes pasos._

```
git clone https://github.com/KiraLei/tienda.git
```

```
npm install
```
```
npm run start
```
